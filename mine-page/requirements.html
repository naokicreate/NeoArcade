Spec

お使いのコンピューターでMineを快適にプレイするための推奨システム要件です。

最小システム要件

OS Windows 10 (64-bit)以上
プロセッサー Intel Core i3-3220 / AMD FX-4300 以上
メモリ 16 GB RAM
グラフィックス NVIDIA GeForce GTX 1660Ti
ストレージ 20 GB 以上の空き容量
その他 インターネット接続（初回インストール時）

推奨システム要件

OS Windows 10/11 (64-bit)
プロセッサー Intel Core i5-6600 / AMD Ryzen 5 1600 以上
メモリ 8 GB RAM
グラフィックス NVIDIA GeForce GTX 2080SP / AMD Radeon RX 560
ストレージ 16 GB 以上の空き容量（SSD推奨）

対応入力デバイス

  • キーボード・マウス
  • Xbox コントローラー
  • その他DirectInput対応コントローラー

※ グラフィック設定を調整することで、さらに低スペックのPCでも動作する可能性があります。

※ スペック対応の詳細については、サポートページをご確認ください。
